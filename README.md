## Sobre mim

Minha jornada na tecnologia começou durante a pandemia, quando tive contato com disciplinas voltadas para tecnologia no curso de Administração da UFRJ. Foi nesse período que descobri meu interesse por desenvolvimento de software e decidi direcionar minha carreira para a área de tecnologia.

Hoje atuo no desenvolvimento de aplicações web, buscando criar soluções que unam tecnologia, usabilidade e impacto real para usuários e empresas. Tenho interesse especial por Inteligência Artificial, UX/UI, Computação em Nuvem e Desenvolvimento Full-Stack.
## 🎯 Objetivos

Busco oportunidades para desenvolver soluções inovadoras, aprender continuamente e colaborar em projetos que gerem valor real para pessoas e negócios.
